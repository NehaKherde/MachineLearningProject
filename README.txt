We have divided the code into folders of-
Generation - The code to pertaining to the programs for generation of synthetic corpora.
Preprocessing - Contains programs for analysis of corpus and preprocessing
Feature Transformation- Contains all relevant programs for forming feature dimensions and conversion of text into vectors
Classification Models- Programs of different classification models used.


Requirements-
Majority of the program are written in Python 2.7.6 
-except Classification_Models\non_linear_svm.py, Classification_Models\ensemble.py written in Python 3.6
Libraries-
Keras 2.2.2
Spacy 2.0.16
Sklearn
numpy 1.15.3
pandas 0.23.4
h5py 2.8.0  


Note: To run the code all files should be in one directory
